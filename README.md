# Blockchain-Based Electronic Voting System

This project presents a blockchain-based electronic voting system developed in Java.

The system demonstrates how blockchain concepts can be used to ensure secure, transparent, and verifiable electronic voting.

## Features
- Blockchain-based vote storage
- Vote validation and integrity control
- Basic client-server architecture
- Use of cryptographic concepts (RSA, PKI)
- Focus on transparency and voter privacy

- Run the main class to start the application

## Notes

This project focuses on demonstrating core concepts of secure electronic voting using blockchain.

It is intended as a conceptual and educational implementation rather than a production-ready system.
